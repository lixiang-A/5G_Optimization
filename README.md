# 5G Optimization Project

## Project goal
This project is used for 5G optimization modeling, reinforcement-learning-based solving, and paper-oriented output for the contest tasks.

## Current repository structure
- 背景信息/
  - contest statement
  - appendix
  - modeling markdown file
  - references
  - earlier modeling materials
- channel_data等2个文件/
  - data for Question 1
  - txt notes
  - initial solving script
- channel_data等2个文件(1)/
  - data for Question 2
  - txt notes
- BS2等5个文件/
  - data for Question 3
  - related readme notes

## Current working principles
- Model quality comes first.
- Reinforcement learning must be meaningfully tied to the contest problem.
- Data, parameters, and assumptions must not be invented without support.
- Outputs should be organized by sub-question.

## Current task focus
1. Check and strengthen the modeling logic for Questions 1 to 3.
2. Make the RL formulation rigorous, explainable, and suitable for undergraduate research writing.
3. Improve code implementation so that results are correct, stable, and reviewable.
4. Prepare outputs that can be integrated into later paper writing.
